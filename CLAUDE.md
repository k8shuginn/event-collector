# 응답
- 모든 응답은 한국어로 진행한다.
- 프로젝트 구조를 파악하라는 요청이 있을 시 모든 파일의 구조를 파악하고 CLAUDE_STRUCT.md 파일에 작성한다.

# 코드 작성 방식
- 작업의 단계는 CLAUDE_PLAN.md 파일에 기록한다.
- 작업이 완료되면 CLAUDE_PLAN.md 파일을 업데이트 한다.
- 모든 .go 파일에는 패키지에 대한 설명을 상단에 영어로 effective go 형식으로 주석을 작성한다.
- 모든 .go 파일에는 메서드에 대한 설명을 메서드 상단에 주석으로 작성한다.
- 메서드 주석은 Go 공식 godoc 형식을 따른다.
  - 첫 줄은 반드시 `// 메서드명`으로 시작하고, 파라미터·리턴값은 산문(prose)으로 이어서 설명한다.
  - 형식:
    ```
    // MethodName param1을 받아 something을 수행하고 result를 반환한다.
    // 실패하면 error를 반환한다.
    ```

# 코드 리팩토링
- 구조를 변경하기 전에 먼저 CLAUDE_STRUCT.md 파일에 변경된 프로젝트의 구조를 작성한다.
